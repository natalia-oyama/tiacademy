1. É verdade que todo arquivo CSS precisa estar dentro de outra pasta para que o navegador possa 
renderizar os estilos das propriedades CSS?
Não, pode-se ter um arquivo de estilo fora de uma pasta. Para manter os arquivos organizados, é 
legal separá-los.

2. É verdade que todo elemento HTML possui por padrão elementos filhos?
Não, alguns elementos podem ter um ou outro elemento filho, mas nem todos terão.
